★ 중요 : html(document)에서 src를 통해 javascript를 가져옴 >> browser가 documnet 접근 가능 >> js document 접근 가능 - document : 브라우저에서 가장 핵심이 되는 object (HTML을 가리키는 object)
> 즉. javascript와 html은 이미 연결되어 있다!
> ★ document 객체에 어떤 정보가 담겨있는지 조사해보기

- javascript는 특정 태그를 가져올 때, 해당 html만 가져오는게 아니라 html을 표현하는 해당 obejct를 가져오는 것.
> javascript의 함수를 사용해 html의 element를 가져온다면, html을 바꿀 수 있게 된다.

- querySelector : elemeent를 CSS 방식으로 검색할 수 있다.
> 여러개가 있는 경우, 제일 첫번째의 값만 나온다.
> 여러개를 받으려면 querySelectorAll을사용해야 한다.
> 특정 순서의 것을 찾으려면 querySelector(".class tag:first-child") 형식으로 사용한다.
> ★ document.querySelector("#id") === document.getElementById("id")

- window : title, head, body 와 관련된 event를 적용할 떄 사용
> 그 외의 경우, document 내의 특정 element에 적용할때는 querySelector, getElementById를 사용
